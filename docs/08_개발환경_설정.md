## 🚀 실행 순서

### 1단계: 프로젝트 클론

```bash
git clone https://github.com/your-username/moneylog.git
cd moneylog
```

### 2단계: Android Studio에서 열기

1. Android Studio → `File → Open` → 프로젝트 폴더 선택
2. Gradle Sync 완료 대기

### 3단계: 실행

- **에뮬레이터**: API 26+ 기기 생성 → Run
- **실기기**: USB 디버깅 활성화 → 연결 → Run

> [!tip] 확인
> 앱이 실행되면 최초 PIN 설정 화면이 표시된다.

## ⚙️ Google API 설정

### Google Cloud Console

1. [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트 생성
2. **Google Drive API** 활성화
3. **OAuth 2.0 클라이언트 ID** 생성 (Android 유형)
   - 패키지 이름: `com.moneylog`
   - SHA-1 지문: 
     ```bash
     keytool -keystore ~/.android/debug.keystore -list -v -alias androiddebugkey -storepass android
     ```
4. 동의 화면 구성 (scope: `drive.appdata`)

### Gemini Nano 설정

1. `build.gradle.kts`에 Google AI Edge SDK 의존성 추가
2. `AndroidManifest.xml`에 AICore 관련 메타 데이터 추가:
   ```xml
   <meta-data
       android:name="com.google.android.aicore"
       android:value="gemini-nano" />
   ```
3. 런타임에 `isAvailable()` 체크 → 미지원 시 AI 기능 숨김

## 🚀 빌드 & 배포

### Debug APK 빌드

```bash
./gradlew assembleDebug
# 출력: app/build/outputs/apk/debug/app-debug.apk
```

### Release AAB 빌드 (Play Store)

```bash
./gradlew bundleRelease
# 출력: app/build/outputs/bundle/release/app-release.aab
```

### 서명 설정 (`app/build.gradle.kts`)

```kotlin
android {
    signingConfigs {
        create("release") {
            storeFile = file(System.getenv("KEYSTORE_PATH") ?: "keystore.jks")
            storePassword = System.getenv("KEYSTORE_PASSWORD") ?: ""
            keyAlias = System.getenv("KEY_ALIAS") ?: ""
            keyPassword = System.getenv("KEY_PASSWORD") ?: ""
        }
    }
}
```

### Play Store 배포

1. [Google Play Console](https://play.google.com/console/) 개발자 등록 ($25 일회성)
2. 앱 생성 → AAB 업로드 → 내부 테스트 → 공개
