## 💾 로컬 데이터베이스

| 구분 | 기술 | 버전 | 설명 |
|---|---|---|---|
| **ORM** | Room | 2.7.1 | Android 공식 SQLite ORM (DAO + Entity) |
| **SQLite** | Android 내장 SQLite | - | 네이티브 SQLite 엔진 |
| **마이그레이션** | Room Auto Migration | - | `@Database(version = N)` + `autoMigrations` |
| **암호화** | SQLCipher (선택) | - | DB 파일 암호화 (필요 시) |
| **설정 저장** | SharedPreferences | - | 앱 설정, 언어, 테마 등 경량 데이터 |
| **java.time** | Core Library Desugaring | 2.1.5 | java.time.LocalDate 등 API 26 이하 지원 |

> [!info] Room + Java 패턴
>
> - Entity: public field POJO — `@Entity`, `@PrimaryKey(autoGenerate = true)`
> - DAO: 인터페이스 반환 타입은 `LiveData<>` (쿼리) / `void`·`long` (쓰기, Executor에서 실행)
> - Repository: `Executors.newSingleThreadExecutor()` 로 DB 쓰기 백그라운드 처리
> - ViewModel: `Transformations.switchMap(selectedYearMonth, ...)` 로 반응형 LiveData 파생

## ☁️ 외부 서비스

| 구분 | 기술 | 설명 |
|---|---|---|
| **백업** | Google Drive API v3 | appDataFolder에 DB 파일 백업·복원 |
| **인증** | Google Sign-In (Credential Manager) | Google 계정으로 Drive 접근 권한 획득 |
| **광고 (추후)** | Google AdMob | 배너 + 인터스티셜 광고 수익화 |

### Google API 설정

| 항목 | 값 |
|---|---|
| OAuth Scope | `https://www.googleapis.com/auth/drive.appdata` |
| 접근 범위 | 앱 전용 폴더만 (사용자 파일 접근 불가) |
| Client Type | Android |
| SHA-1 지문 | `keytool -keystore ~/.android/debug.keystore -list` |
