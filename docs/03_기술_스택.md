# 03. 기술 스택

> **최종 업데이트**: 2026-04

---

## 📱 Android UI

| 구분 | 기술 | 버전 | 설명 |
|---|---|---|---|
| **언어** | Kotlin | 2.x | Android 공식 개발 언어 |
| **UI 프레임워크** | Jetpack Compose | 최신 BOM | 선언적 UI 툴킷 |
| **Material Design** | Material 3 | 최신 | Google Material You 디자인 시스템 |
| **네비게이션** | Navigation Compose | 최신 | 화면 전환 및 Back Stack 관리 |
| **차트** | Vico | 최신 | Compose 네이티브 차트 라이브러리 |
| **DI** | Hilt | 최신 | Dagger 기반 의존성 주입 |
| **비동기** | Kotlin Coroutines + Flow | 최신 | 비동기 처리 및 반응형 스트림 |
| **아이콘** | Material Icons Extended | 최신 | Material 아이콘 팩 |

---

## 💾 로컬 데이터베이스

| 구분 | 기술 | 버전 | 설명 |
|---|---|---|---|
| **ORM** | Room | 최신 | Android 공식 SQLite ORM (DAO + Entity) |
| **SQLite** | Android 내장 SQLite | - | 네이티브 SQLite 엔진 (WASM 불필요) |
| **마이그레이션** | Room Auto Migration | - | `@Database(version = N)` + `autoMigrations` |
| **암호화** | SQLCipher (선택) | - | DB 파일 암호화 (필요 시) |
| **설정 저장** | EncryptedSharedPreferences | - | PIN 해시 등 민감 데이터 암호화 저장 |

> [!info] 왜 Room인가?
> - Android 네이티브 SQLite ORM으로 컴파일 타임 SQL 검증
> - DAO 인터페이스로 타입 안전한 쿼리
> - LiveData/Flow 연동으로 UI 자동 업데이트
> - 자동 마이그레이션 지원

## ☁️ 외부 서비스

| 구분 | 기술 | 설명 |
|---|---|---|
| **백업** | Google Drive API v3 | appDataFolder에 DB 파일 백업·복원 |
| **인증** | Google Sign-In (Credential Manager) | Google 계정으로 Drive 접근 권한 획득 |
| **광고 (추후)** | Google AdMob | 배너 + 인터스티셜 광고 수익화 |

### Google API 설정

| 항목 | 값 |
|---|---|
| OAuth Scope | `https://www.googleapis.com/auth/drive.appdata` |
| 접근 범위 | 앱 전용 폴더만 (사용자 파일 접근 불가) |
| Client Type | Android |
| SHA-1 지문 | `keytool -keystore ~/.android/debug.keystore -list` |

## 🏗️ 배포 & 인프라

| 구분 | 기술 | 설명 |
|---|---|---|
| **배포** | Google Play Store | APK/AAB 배포 |
| **빌드** | Gradle (Kotlin DSL) | Android 공식 빌드 시스템 |
| **CI/CD** | GitHub Actions | main 브랜치 push → 자동 빌드·Play Store 업로드 |
| **최소 SDK** | API 26 (Android 8.0) | Gemini Nano는 API 31+ 권장 |
| **Target SDK** | API 35 (Android 15) | 최신 타겟 |

> [!tip] 서버 비용 제로
> 네이티브 앱이므로 서버 호스팅 비용 없음. Play Store 개발자 등록비($25 일회성)만 필요.
