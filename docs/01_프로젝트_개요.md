# 01. 프로젝트 개요

> **최종 업데이트**: 2026-04-17
> [!info] 프로젝트 목표
> 개인 또는 가정의 수입·지출을 손쉽게 기록하고, 카테고리별 분석을 통해 건전한 소비 습관을 만들 수 있는 **안드로이드 네이티브 가계부 앱**.
> 서버 없이 기기 내 Room(SQLite)에 데이터를 저장하고, Google Drive 백업·복원과 **Gemini Nano 온디바이스 AI 요약** 기능을 지원한다.

---

## 🎯 핵심 가치 제안

- **안드로이드 네이티브**: Java 17 + XML Layouts(ViewBinding)으로 구현하는 오프라인 완전 동작 앱
- **간편한 수입·지출 기록**: 최소한의 입력으로 빠르게 거래를 등록
- **자동화**: 고정 수입·지출(월급, 구독료 등)을 예약일에 자동 등록
- **Google Drive 백업**: 클라우드 백업으로 데이터 안전 보장, 기기 교체 시 복원
- **카테고리 기반 분석**: 항목별 자동 분류 & 통계
- **Gemini Nano AI 요약**: 온디바이스 AI로 월별·카테고리별 소비 패턴을 자연어로 요약
- **광고 기반 수익화**: 추후 Google AdMob 배너 광고 통합

## 📐 핵심 원칙

1. **안드로이드 전용**: Java 17 + XML Layouts 네이티브 앱
2. **로컬 퍼스트**: 데이터는 기기 내 Room(SQLite)에 저장, 네트워크 없이도 사용 가능
3. **온디바이스 AI**: Gemini Nano로 인터넷 없이 소비 패턴 분석·요약
4. **데이터 주권** — 사용자 데이터는 자기 기기 + 자기 Google Drive에만 존재
5. **점진적 수익화** — MVP 완성 후 광고 레이어 추가
