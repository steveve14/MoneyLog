# 07. UI 화면 설계 (MoneyLog) > **최종 업데이트**: 2026-04 ## 🗺️ 화면 구조 (네비게이션) - PIN Lock -> Dashboard - Dashboard -> Transactions, Budget, Statistics, Settings - Transactions -> Detail/Edit, Add - Statistics -> AI Summary, Category Analysis, Monthly Trend - Settings -> Recurring Mgmt, Category Mgmt, Backup, PIN Change ## 📱 주요 화면 명세 ### 1. PIN 잠금 화면 - PIN 입력 (4자리) 및 생체인증 지원. ### 2. 대시보드 (메인) - 월별 수입/지출/잔액 요약 카드. - 자동 등록된 내역 알림. - Gemini Nano 기반 AI 한줄 요약. - 카테고리별 지출 도넛 차트. - 최근 거래 내역 (Top 3). - 하단 네비게이션 (홈, 내역, 등록, 통계, 설정). ### 3. 거래 등록 - 지출/수입 탭. - 금액, 날짜, 카테고리(AI 추천 포함), 결제 수단, 메모 입력. ### 4. 거래 내역 - 월별 목록, 필터링(전체/지출/수입), 검색. - [자동] 배지 표시. ### 5. 고정 수입·지출 관리 - 활성/비활성 상태 관리 및 주기적 거래 목록. ### 6. 통계 + AI 요약 - Gemini Nano 기반 심층 소비 분석 카드. - 최근 6개월 지출 추이 (라인 차트). - 카테고리별 지출 바 차트 및 전월 대비 증감률. ### 7. 예산 관리 - 전체 예산 및 카테고리별 예산 프로그레스 바. - 초과 경고 표시. ### 8. 설정 - 데이터 관리, 백업/복원(Google Drive), AI 상태, 보안, 외관(다크모드), 데이터 초기화. ## 🎨 디자인 가이드 - **Primary:** Indigo Blue (#4F46E5) - **Income:** Green (#10B981) - **Expense:** Red (#EF4444) - **Warning:** Amber (#F59E0B) - **AI/Gemini:** Cyan (#06B6D4) - **Font:** Pretendard - **Components:** Material 3 (Cards, FAB, NavigationBar, etc.)
